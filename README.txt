The ZIP file contains only the code files. The necessary data to run the code must be downloaded from the following Google Drive folder and placed in the same folder as the .py/.ipynb files:

[https://drive.google.com/drive/folders/1vx4mmlMwV-Nb4JGHfiPWgw0OIowkoHOW?usp=sharing](https://drive.google.com/drive/folders/1vx4mmlMwV-Nb4JGHfiPWgw0OIowkoHOW?usp=sharing)

The script files (.py/.ipynb) should be run section by section (i.e., section 1. and 2.), without going back or skipping ahead; otherwise, the correct functioning of the scripts cannot be guaranteed.

The scripts for the project are:

1. **Missing Data, Visualization, and Outlier Detection:**
   - missing_data.py
   - features_visualization_outlier_removal.ipynb

2. **Clustering:**
   - features_clustering.ipynb
   - Skeletons_clustering.ipynb
   - Augmented_data_clustering.ipynb

**Auxiliary Scripts:**
- extract_images.py
- draw_pose.py
- empca.py
- visualization_dimensionality_red_functions.py
- outlier_detection_functions.py
- skeletons_aux_functions.py
- clustering_functions.py
- augmentation_script.py
